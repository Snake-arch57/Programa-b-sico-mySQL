# Programa básico mySQL
## Neste projeto desenvolvi um programa básico para mySQL, com proteção contra sql Injection.
### No projeto fiz uma tipo de local empresarial onde o ADM da empresa iria operar o sistema de forma local. Colocando metas, adicionando funcionários e excluindo funcionários.
### O projeto tem um código para criar as tabelas e insere dados aleatórios na mesma
